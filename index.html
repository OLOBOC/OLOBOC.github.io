Caso de Estudio - DAM

Monetix: Gestión Financiera Inteligente

Resumen del Proyecto

Monetix es una plataforma premium concebida para unificar el control de gastos cotidianos con la gestión analítica de carteras de inversión. Incorpora análisis automatizados e interactivos por medio de IA para facilitar la toma de decisiones financieras.

Arquitectura y Tecnologías

  • Frontend: Ionic + Angular (TypeScript). Estructura SPA altamente optimizada y fluida. Interfaz basada en Glassmorphism con carga perezosa (lazy loading) de módulos y animaciones aceleradas por hardware.
  • Backend: PHP Puro desarrollado con un patrón de arquitectura Modelo-Vista-Controlador (MVC) a medida, asegurando el desacoplamiento de la lógica de negocio, enrutamiento seguro de peticiones y saneamiento de datos.
  • Base de Datos: MySQL. Modelo de datos relacional para usuarios, cuentas bancarias, transacciones recurrentes, tipos de activos financieros y registros de cotizaciones históricas.
  • Integración de IA: Groq Cloud API y el modelo Llama 3.1. Permite realizar consultas contextuales al Asesor Financiero, analizando en tiempo real la salud de la cartera del usuario de forma totalmente anónima y segura.

Características Técnicas Clave

  • Seeding Inteligente: Sistema que genera automáticamente un historial de transacciones financieras aleatorias y realistas de hasta un año completo, facilitando las pruebas de gráficos y reportes.
  • Reportes PDF: Generación y formateo dinámico de informes de rendimiento y estados contables desde el servidor, listos para descargar o imprimir.
  • Seguridad de Datos: Hash de contraseñas robusto, gestión segura de sesiones de usuario y variables de entorno protegidas.
Caso de Estudio - DAW

Estilo Vivo: Gestión de Peluquería

Resumen del Proyecto

Estilo Vivo es un sistema web completo desarrollado en Laravel diseñado para digitalizar la agenda y reservas de una peluquería, eliminando la gestión telefónica manual y mejorando la ocupación diaria.

Arquitectura y Tecnologías

  • Backend: Laravel 11. Aprovecha las ventajas del framework: inyección de dependencias, Eloquent ORM para consultas ágiles a la base de datos, migraciones estructuradas y sistema de middleware para el control de roles de usuario.
  • Autenticación: Laravel Breeze. Implementación rápida, limpia y segura de los flujos de registro, login, verificación de correo y recuperación de contraseñas.
  • Frontend: Blade (motor de plantillas de Laravel) acoplado con Alpine.js para la interactividad rápida del lado del cliente (modales, menús y selects reactivos) y estilado con Tailwind CSS.
  • Compilación: Vite. Configuración ágil de empaquetado para asegurar tiempos de carga mínimos.

Flujos de Trabajo Implementados

  • Rol Cliente: Registro/Login -> Navegación por catálogo de servicios -> Selección de peluquero preferido -> Selector visual de fechas y horas disponibles -> Confirmación instantánea y guardado en su panel personal.
  • Rol Peluquero: Panel de control optimizado. Carga una agenda diaria ordenada cronológicamente con los servicios pendientes, duración, nombre del cliente y notas adicionales.
